🚀 LogicTech Messenger: The "Zero-to-Profit" Architecture
Welcome to the LogicTech Messenger repository. This isn't just another chat app; it’s a high-performance, full-stack communication engine built with the React + Next.js + TypeScript stack. It’s designed for speed, persistence, and that "vibe coding" aesthetic we love.

🛠 What’s Under the Hood?
We didn’t just build a UI; we built an ecosystem. Here’s the "A to Z" of what we’ve implemented:

Real-Time Orchestration: Powered by Socket.io for instantaneous message delivery and global state synchronization.

Presence Engine: A live "Active Users" tracking system so you never talk to a ghost.

Typing Intelligence: Real-time debounced typing indicators to make the conversation feel alive.

Auth & Security: Robust JWT (JSON Web Token) authentication paired with Bcrypt password hashing—because security is non-negotiable.

Persistence Layer: MongoDB Atlas integration ensures your message history lives forever (or at least as long as the database does).

Session Immortality: LocalStorage integration that keeps users logged in and maintains their theme preferences across refreshes.

The "Vibe" UI: A sleek, Tailwind CSS-powered interface featuring a dynamic Dark Mode for late-night coding sessions.

🚀 Key Features
✅ Global Chat: A unified channel for real-time collaboration.

✅ Read Receipts: Visual confirmation (✓✓) once messages are processed by the server.

✅ Emoji & Sticker Lab: Full emoji picker integration and a custom sticker suite for high-ticket expression.

✅ Cross-Platform Readiness: Fully responsive design tested on Ubuntu Linux and optimized for HP EliteBook performance.

🏗 Setup & Deployment
1. The Frontend (/client)
npm install
npm run dev
2. The Engine (/server)
node server.js
